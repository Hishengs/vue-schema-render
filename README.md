# vue-schema-render
A Form Render Based on JSON Schema for Vue

## Features

- 内置 ElementUI 组件
- 支持自定义组件
- 支持自由布局
- 支持样式覆盖
- 支持多语言
- 支持列表组件拖拽
- 支持无限嵌套表单以及嵌套校验，自定义校验规则
- 支持组件联动
- 详尽的文档及示例


## 安装
`npm install -S vue-schema-render`
